Create a desktop shortcut for Obsidian Todo:

1. Right-click on desktop → New → Shortcut
2. Location: C:\Users\info\AppData\Local\nvim\bin\obsidian-todo-simple.bat
3. Name: Obsidian Todo
4. Finish
5. Right-click shortcut → Properties → Change Icon → Choose an icon
6. You can also set "Run: Minimized" if you prefer

This will:
- Open a new PowerShell window
- Navigate to your Obsidian folder  
- Create todo.md if it doesn't exist
- Open todo.md in Neovim

Alternative: You can pin this shortcut to:
- Taskbar (right-click → Pin to taskbar)
- Start menu (drag to Start button)
