You can create a desktop shortcut for this:

1. Right-click on desktop → New → Shortcut
2. Location: C:\Users\info\AppData\Local\nvim\bin\obsidian-today.bat
3. Name: Obsidian Today
4. Finish
5. Right-click shortcut → Properties → Change Icon → Choose an icon

Alternatively, you can pin it to taskbar or start menu.

Usage:
- In Neovim: <leader>oe
- Desktop shortcut: Double-click the icon
- Both will open a new PowerShell terminal with today's journal note loaded in Neovim
